<h1 align="center"> Vacation Planner </h1>

<p align="center">

</p>

## Tecnologias Utilizadas

- ReactJS ( searchParams, URLState )
- TypeScript
- Tailwindcss
- Tailwind Merge
- Shadcn Ui
- Lucide React
- React Helmet Async
- React Router DOM
- React Query
- React Hook Form
- Zod
- Date Fns
- Sonner
- Eslint (simple-import-sort)
- Prettier (prettier-plugin-tailwindcss)
- Vite
